# MARDAKHAY — AI & Data Science Engineering

A high-precision landing page for an AI and data engineering firm, built with semantic HTML, vanilla CSS, and vanilla JS — no frameworks, no dependencies.

## Features

- Custom animated cursor with hover state
- Scroll-driven progress bar and section-active nav highlighting
- Scroll reveal animations on every section
- Animated counters in the metrics bar
- Bento card mouse-follow spotlight effect
- Terminal typing animation on M-Cloud card
- Inline SVG cybersecurity matrix in Intelligence Audit card
- Seamless marquee partner strip
- Hero glitch flicker and orb parallax on mouse move
- Fully responsive across 7 breakpoints (320 px → 1920 px+)
- Respects `prefers-reduced-motion`
